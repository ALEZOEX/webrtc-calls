# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-10

### Added
- Complete video conferencing functionality with WebRTC
- Screen sharing capabilities
- Interactive whiteboard with drawing tools
- Real-time chat with emoji support
- Responsive UI with modern design
- Docker support for easy deployment
- Environment-based configuration management

### Changed
- Improved UI/UX design with a modern dark theme
- Enhanced video grid layout for better participant visibility
- Optimized WebSocket connections for better performance
- Improved error handling and user feedback

### Fixed
- Various bug fixes related to peer connections
- Issues with screen sharing functionality
- Whiteboard synchronization between participants
- Chat message history management

## [1.0.0] - 2025-10-15

### Added
- Initial release with basic video conferencing
- Simple chat functionality
- Basic room management system
